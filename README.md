# Figure Radar

Figure Radar is an open-source figure discovery and shop comparison platform.

The project organizes collectible figures using a structured model:

Work → Character → Manufacturer → Product Group → Product Variant → Shop Listing

Users can search figures, compare listings from multiple shops, and navigate directly to external retailers.

Figure Radar does not process payments or operate as a marketplace.
Its goal is to improve discoverability, data quality, and price comparison across the figure collecting ecosystem.

## Current Features

- Figure search
- Alias search
- Product detail pages
- Multi-shop listing support
- Click tracking
- Admin dashboard
- Quick Add workflow
- Bulk CSV import
- Review Queue
- Image management
- Shop comparison UI
